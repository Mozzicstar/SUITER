1) Start services

- API (mock server):
  In a terminal run:
    python3 api_mock.py
  This initializes the sample DB and starts the API on http://localhost:3000

- Frontend (static):
  In a second terminal run:
    ./frontend/server.sh 8080
  Then open http://localhost:8080 in your browser

- Optional: Live demo on GitHub Pages
